# E-commerce Log In

Authentication and Authorization system for a e-commerce.

# Getting Started

## Install dependencies

```
$ npm install
```

## Deploy the project

```
$ npm run dev
```

# Build with

- Node.js
- Express.js
- Node mailer
- Cookie Parser
- Nodemon
- JWT
- Bcrypt
- Mongoose
- Passport
