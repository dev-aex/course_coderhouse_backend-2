# E-commerce Log In

Authentication and Authorization system for a e-commerce.

# Getting Started

## Install dependencies

```
$ npm install
```

## Deploy the project

```
$ npm run dev
```

# Build with

- Node.js
- Express.js
- Handlebars
- Nodemon
- JWT
- MongoDB
- Passport
